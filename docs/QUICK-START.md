# ⚡ Guía Rápida - LaComarca GastroPark

Una guía concisa para comenzar rápidamente con el sistema.

---

## 🔐 Paso 1: Iniciar Sesión

### Opción A: Como Administrador
```
Email:      admin@gmail.com
Contraseña: admin
```

### Opción B: Como Gerente
```
Email:      leskas@gmail.com
Contraseña: local
```

1. Abre el navegador
2. Ve a la URL del proyecto
3. Ingresa tu email y contraseña
4. Haz clic en "Iniciar Sesión"

---

## 👨‍💼 SI ERES ADMINISTRADOR

### Rápida Tarea: Crear un Usuario

```
1. Menú Lateral → Gestionar Usuarios
2. Botón "➕ Nuevo Usuario"
3. Completa el formulario:
   - Nombre: Juan Pérez
   - Email: juan@ejemplo.com
   - Contraseña: MiPass123
   - Rol: Gerente
4. Haz clic en "✔️ Crear Usuario"
```

### Rápida Tarea: Crear un Evento

```
1. Menú Lateral → Gestionar Eventos
2. Botón "➕ Nuevo Evento"
3. Completa:
   - Nombre: Cena Especial
   - Fecha: 2025-03-15
   - Hora: 19:30
   - Descripción: Una noche memorable
   - Imagen: (opcional)
4. Haz clic en "✔️ Guardar Evento"
```

---

## 🏪 SI ERES GERENTE

### Rápida Tarea: Registrar un Producto

```
1. Menú Lateral → Gestionar Productos
2. Botón "➕ Nuevo Producto"
3. Completa:
   - Nombre: Filete de Res
   - Categoría: Platos Principales
   - Precio: 25.50
   - Estado: Disponible
   - Descripción: Filete jugoso... (opcional)
   - Foto: Selecciona una imagen
4. Haz clic en "✔️ Crear Producto"
```

### Rápida Tarea: Agregar Fotos al Producto

```
1. En el listado de productos
2. Busca tu producto
3. Haz clic en "📸 Galería"
4. Arrastra fotos o haz clic en "➕ Agregar Fotos"
5. Se cargarán automáticamente
6. Haz clic en "⭐ Establecer como principal" en la foto que prefieras
```

### Rápida Tarea: Editar Precio

```
1. Busca el producto en el listado
2. Haz clic en "✏️ Editar"
3. Cambia el campo "Precio"
4. Haz clic en "✔️ Guardar Cambios"
```

## ❓ Problemas Frecuentes

### No puedo subir una foto
```
❌ La foto es muy grande (> 2MB)
✅ Comprime la foto con herramientas online

❌ El formato es incorrecto
✅ Usa JPG o PNG

❌ No tengo conexión
✅ Verifica tu conexión a internet
```

### El producto no aparece
```
❌ Está marcado como "No disponible"
✅ Cámbialo a "Disponible"

❌ Hay un error de validación
✅ Revisa que todos los campos obligatorios estén completos
```

### No puedo editar
```
❌ No tienes permisos
✅ Comprueba tu rol (Admin vs Gerente)

❌ Hay un error del sistema
✅ Recarga la página (F5)
```
