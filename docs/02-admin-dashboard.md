# Dashboard del Administrador

## Descripción General

El dashboard del administrador proporciona una visión completa del sistema con acceso a todas las funcionalidades de gestión.

---

## Vista Principal del Dashboard

Al iniciar sesión como administrador, verás:

### 📊 Tarjetas de Estadísticas
- **Total de Usuarios:** Cantidad total registrada
- **Usuarios Activos:** Usuarios con acceso vigente
- **Eventos Activos:** Eventos programados y confirmados

### 🔗 Enlaces Rápidos
Acceso directo a:
- Gestionar Usuarios
- Gestionar Eventos

### 📊 Gráficos (en desarrollo)
- Actividad del sistema
- Estadísticas de usuarios
- Tendencias de eventos

---

## Navegación Principal

El menú lateral te permite acceder a:

1. **👤 Gestionar Usuarios** → [Ver guía](03-gestionar-usuarios.md)
2. **📅 Gestionar Eventos** → [Ver guía](04-gestionar-eventos.md)
3. **⚙️ Configuración del perfil**

---

## Características del Administrador

✅ **Acceso completo** a todos los módulos  
✅ **Crear nuevos usuarios** con diferentes roles  
✅ **Editar información** de usuarios, eventos 
✅ **Eliminar registros** (con confirmación de seguridad)  
✅ **Cambiar estados** de eventos
